# BodyPainting - Plataforma de Gestión Comercial

Aplicación web Full Stack desarrollada utilizando Spring Boot y React que permite la gestión integral de productos, pedidos, clientes y vendedores dentro de una plataforma de comercio electrónico.

El proyecto fue desarrollado bajo metodología Scrum, aplicando arquitectura por capas, autenticación basada en JWT y buenas prácticas de desarrollo colaborativo utilizando Git y GitHub.

---

## Características principales

### Gestión de Usuarios

- Registro de clientes
- Inicio de sesión seguro mediante JWT
- Gestión de roles:
  - Administrador
  - Vendedor
  - Cliente

### Gestión de Productos

- Alta de productos
- Baja lógica de productos
- Importación de imágenes
- Control de stock
- Alertas de stock mínimo
- Gestión de kits de productos

### Experiencia de Compra

- Catálogo de productos
- Carrito de compras
- Confirmación de pedidos
- Gestión de domicilios de envío
- Aplicación de cupones de descuento

### Gestión Comercial

- Visualización de pedidos
- Actualización de estados
- Reportes de productos más vendidos
- Reportes de stock mínimo
- Asistente inteligente de reposición

### Notificaciones

- Confirmación automática de pedidos
- Notificaciones por correo electrónico
- Comunicación automática de cambios de estado

## Metodología de Trabajo

El proyecto fue desarrollado utilizando Scrum.

Durante tres sprints se implementaron funcionalidades incrementales incluyendo:

- Gestión de usuarios
- Gestión de productos
- Carrito y pedidos
- Seguridad JWT
- Reportes comerciales
- Sistema de cupones
- Notificaciones automáticas

La planificación y seguimiento se realizó mediante Jira.
